# Loan-Approval-Prediction
# 📊 Loan Approval Prediction using Logistic Regression in Orange Data Mining

## 📌 Project Description

This project demonstrates how to build a **Loan Approval Prediction Model** using **Logistic Regression** in **Orange Data Mining**, a no-code machine learning platform.

The objective is to predict whether a loan application will be **Approved** or **Rejected** based on applicant financial and demographic information.

## 📊 Dataset Information

### Input Features

| Feature | Description |
|----------|------------|
| Gender | Applicant Gender |
| Income | Monthly Income |

| LoanAmount | Requested Loan Amount |
| Dependents | Number of Dependents |
| Education | Education Level |
| Self_Employed | Employment Status |
| Property_Area | Urban/Semiurban/Rural |

### Target Variable

| Variable | Description |
|----------|------------|
| Loan_Status | Approved / Rejected |

## 🚀 Steps Performed

### Step 1: Import Dataset
- Open Orange Data Mining
- Add the **File** widget
- Import the loan approval dataset

### Step 2: Select Columns
- Use **Select Columns**
- Set:
  - Features → Input Variables
  - Loan_Status → Target Variable

### Step 3: Apply Logistic Regression
- Connect the **Logistic Regression** widget
- Train the classification model

### Step 4: Evaluate Model
- Connect **Test & Score**
- Analyze model performance

---

## 📈 Logistic Regression

Logistic Regression is a supervised machine learning algorithm used for binary classification.

In this project:

```text
Approved = 1
Rejected = 0
```

The model predicts the probability of loan approval based on applicant information.
